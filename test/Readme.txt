Directory structure for /test:

- config: configuration files for CPAchecker
- expected-results: results from earlier runs to check for regressions
- output: all files produced by CPAchecker during analysis like log file, graphs etc. (not in repository)
- original-sources: unmodified source code from which test cases were made
- scripts: script files for running single tests and test suites
- test-suites: files containing a list of test cases considered to be a test suite
- tests: test cases in a form that CPAchecker can handle
